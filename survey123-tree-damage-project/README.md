# Survey123 Tree Damage Data Collection & Mapping 

## Project Overview
This project demonstrates the design and implementation of a GIS-based field data collection workflow using ArcGIS Survey123. 

## Objective
To create a survey and map its entries.

## Survey Design

The form includes:
- Geopoint location capture
- Categorical fields
- Structured dropdown lists

## Workflow

1. Survey created in Survey123.
2. Submissions collected and stored.
3. Map configured to visualize data.



## Tools Used

- ArcGIS Survey123
- ArcGIS Pro
