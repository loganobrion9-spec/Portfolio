# Coded Adventure Game

## Project Overview
This project is a text-based adventure game built to demonstrate programming logic, user interaction handling, and structured control flow. The game allows users to navigate decision paths that affect outcomes.

## Objective
To design and implement an interactive adventure game using structured programming concepts such as conditionals, loops, and functions.
