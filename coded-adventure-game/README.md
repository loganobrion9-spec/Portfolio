# Coded Adventure Game

## Project Overview
This project is a text-based adventure game built to demonstrate programming logic, user interaction handling, and structured control flow. The game allows users to navigate decision paths that affect outcomes.

## Objective
To design and implement an interactive adventure game using structured programming concepts such as conditionals, loops, and functions.

## Features
- Branching decision paths
- Multiple game outcomes
- Input validation
- Replay functionality
- Structured modular code design

## Tools Used
- Python
- Command line interface

## Skills Demonstrated
- Logical flow control
- Conditional statements
- Functions and modular programming
- User input handling
- Debugging and testing

## Project Structure
- game.py – Main game loop
- gamefunctions.py – Core gameplay mechanics
- save_load.py – Save and load system
- wanderingMonster.py – Random encounter system
- images/ – Game visual assets
- sounds/ – Audio assets


## Requirements
- Python 3.x installed
- pygame

## How to Run
1. Download or clone this repository.
2. Navigate into the project folder:

cd coded-adventure-game

3. Run the main file:

python game.py

## Notes

- Make sure the images and sounds folders remain in the same directory as `game.py`.
- Do not remove or rename supporting modules.
