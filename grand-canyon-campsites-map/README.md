# Grand Canyon Campsites – Static Map

## Project Overview
This project is a designed static map showing the spatial distribution of designated campsites within Grand Canyon National Park that have both trail and riverside access.

## Objective
To create a clean, publication-quality static map suitable for park visitors, planners, or informational materials.

## Methods
- Symbolized campsites using clear, intuitive markers
- Applied scale-dependent labeling
- Designed layout in map view with title, legend, north arrow, and scale bar
- Adjusted color scheme for readability and accessibility

## Tools Used
- ArcGIS Pro
- Layout view design tools
- Symbology configuration

## Skills Demonstrated
- Cartographic design principles
- Static map layout creation
- Map composition and visual balance
- Geographic data management
